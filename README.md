## 📋 Sobre o Projeto

O **FarChain** é uma plataforma digital de rastreabilidade, auditoria, transparência, farmacovigilância e segurança de medicamentos do CEAF, com foco em medicamentos de alto custo distribuídos pelo SUS.

### 🎯 Principais Funcionalidades

- ⛓️ **Blockchain Permissionado** — Ledger SHA-256 imutável para auditoria completa
- 🔍 **Rastreabilidade por QR Code** — Consulta pública segura por lote e unidade
- 💊 **Gestão de Medicamentos CEAF** — Cadastro completo com protocolos clínicos
- 🌡️ **Monitoramento de Cadeia Fria** — Alertas de temperatura para biológicos
- 🏥 **Gestão por Distritos** — 12 Distritos Sanitários de Salvador/BA
- 📊 **Dashboard Analítico** — Indicadores estratégicos em tempo real
- 🔔 **Farmacovigilância** — Registro de RAM, queixas técnicas e desvios
- 🔐 **Segurança Avançada** — RBAC, JWT, Argon2, LGPD compliance

## 🚀 Instalação Rápida (WSL Ubuntu 22.04)

```bash
# Clonar o repositório
git clone https://github.com/seu-usuario/farchain.git
cd farchain

# Executar script de setup
chmod +x scripts/setup.sh
./scripts/setup.sh

# Iniciar o projeto
pnpm dev
```
